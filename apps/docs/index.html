Quick Start

Get up and running in minutes

1. Install packages

# For React projects
npm install @josui/react @josui/tailwind

# For Vue projects
npm install @josui/vue @josui/tailwind

2. Import styles

// In your main CSS file or entry point
import '@josui/tailwind/styles.css';

3. Use components

React
import { Button } from '@josui/react'; function App() { return ( <Button variant="primary"> Click me </Button> ); }
Vue
<script setup> import { Button } from '@josui/vue'; </script> <template> <Button variant="primary"> Click me </Button> </template>
