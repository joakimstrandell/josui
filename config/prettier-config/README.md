# @josui/prettier-config

> Internal package — not published to npm

Shared Prettier configuration for the josui monorepo.

## Usage

In `package.json`:

```json
{
  "prettier": "@josui/prettier-config"
}
```

With Tailwind CSS (includes automatic class sorting):

```json
{
  "prettier": "@josui/prettier-config/tailwind"
}
```
