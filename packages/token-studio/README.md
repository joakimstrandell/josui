# @josui/token-studio

Local CRUD editor for DTCG token category files. Installable CLI + web UI.

## What's inside

```bash
pnpm add -D @josui/token-studio
```

- **CLI** – `josui-token-studio` command with config discovery and options
- **Web UI** – Local browser-based token editor with categories, changes, and settings views

```bash
npx josui-token-studio
```

See [USAGE.md](./USAGE.md) for CLI options, config file format, and routes.
